### How to Create a Virtual Machine

1. Open AQEMU Private
2. Click **"New Virtual Machine"**
3. Enter a name for your VM (e.g., "Windows 11 Test")
4. Select operating system type (Windows, Linux, etc.)
5. Choose RAM amount (2GB minimum)
6. Create virtual disk (20GB recommended)
7. Click **"Finish"** — VM is ready

### How to Install an Operating System

1. Select your created VM
2. Click **"Settings"** → **"Storage"**
3. Browse to your OS ISO file (Windows, Ubuntu, etc.)
4. Click **"OK"**
5. Click **"Start"** to boot the VM
6. Install OS as you would on a real PC

### How to Take a Snapshot

1. Power off or pause your VM
2. Click **"Snapshots"** tab
3. Click **"Take Snapshot"**
4. Name your snapshot (e.g., "After Installation")
5. Restore anytime with one click

### How to Set Up Shared Folders

1. Power off the VM
2. Go to **"Settings"** → **"Shared Folders"**
3. Click **"Add Folder"**
4. Select a folder on your host PC
5. Enable **"Auto-mount"**
6. Start VM → folder appears inside guest OS
